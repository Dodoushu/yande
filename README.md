# yande

一个 [yande](http://yande.re) 的第三方 app .可以下载和查看 图片，并且收藏图片和将 tag 加入快捷搜索栏。

## 预览

![](https://raw.githubusercontent.com/xiao-po/readme_image/master/image/yandeIndexPage.jpg)

## 目录结构

```
|-- android
|   |-- app // Android 代码
|-- lib
|   |-- dao // 数据库相关代码
|   |-- model // model 对象
|   |-- route // 路由 
|   |-- service // service 方法
|   |-- utils // 工具类
|   |-- view // view 
|   `-- widget // 公共组件
`-- test
```
